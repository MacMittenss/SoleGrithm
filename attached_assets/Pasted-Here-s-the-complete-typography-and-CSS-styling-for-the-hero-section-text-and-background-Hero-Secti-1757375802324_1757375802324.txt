Key Typography Features
Font Family: Inter (inherited from body)
Color: Uses CSS custom properties (var(--white) = whitesmoke)
Responsive Scaling: Uses clamp() for fluid typography with viewport units
Letter Spacing: Negative for main title (-0.07vw), positive for small text (0.14vw)
Text Transform: None for main title, uppercase for welcome text
Line Height: Very tight (0.8) for dramatic effect on main title
